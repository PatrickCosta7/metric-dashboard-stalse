# Metric Dashboard (Next.js + TypeScript)

Dashboard simples para exibir métricas e campanhas usando **Next.js (App Router)**, **TypeScript**, **TailwindCSS** e **Recharts**, consumindo dados de uma **API simulada** (rotas do próprio Next.js).

# Link da DEMO:

- https://metric-dashboard-stalse.vercel.app/ 


# Como rodar localmente:

### Requisitos
- Node.js (recomendado: versão LTS)
- npm / yarn / pnpm

## Passo a passo

1) Instale as dependências:
```bash
npm install
```

2) Rode o ambiente de desenvolvimento:
```bash
npm run dev
```

3) Acesse no navegador:
- `http://localhost:3000`

## O que você vai ver
- Simulação de erro
- Tema claro/escuro (e sistema)
- Skeleton ao carregar a página
- Cards com 3 métricas principais
- Tabela de campanhas com filtro por status
- Gráfico de investimento por canal
